Sobre Mim

Sou estudante do 3º semestre de Sistemas de Informação na UNEB, onde construi minha base de lógica de programação atráves da linguagem C e tive contato com Python na Iniciação Científica. Atualmente, estou expandindo meus conhecimentos para área de Front-End com HTML, CSS e JavaScript atráves de cursos online realizados por conta própria e trabalhando em pequenos projetos com Python. Fascinada em entender como as coisas funcionam por baixo de todas as interfaces e lógicas, estudar sobre tecnologia deixa de ser apenas "trabalhoso" e complicado como é normalmente para iniciantes, e se torna um processo de persistências, descobertas e apredizados vindo de erros e mais erros, para que os acertos se tornem mais frequentes, para sentir o entusiasmo pelas novidades e a sensação de satisfação por um trabalho bem feito .
